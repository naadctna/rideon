# 🏍️ FAQ - Sistem Alamat & Koordinasi RideOn

## ❓ Pertanyaan yang Sering Diajukan

### **1. Untuk apa field alamat digunakan?**

**Jawaban:** Field alamat memiliki beberapa fungsi penting:

✅ **Verifikasi Identitas**
- Memastikan data penyewa lengkap dan valid
- Backup informasi untuk keamanan platform

✅ **Koordinasi Lokasi**
- Pemilik motor bisa melihat area umum penyewa
- Memudahkan penentuan titik temu pickup/return
- Estimasi jarak untuk koordinasi

✅ **Keamanan & Trust**
- Mengurangi risiko penipuan
- Data lengkap untuk follow-up jika ada masalah

✅ **Customer Service**
- Admin bisa membantu mediasi dispute
- Backup contact information

---

### **2. Bagaimana sistem pickup motor bekerja?**

**Jawaban:** RideOn menggunakan sistem **koordinasi langsung** antara pemilik dan penyewa:

#### 🔄 **Alur Proses:**

1. **Booking & Approval**
   ```
   Penyewa → Pilih Motor → Booking → Admin Review → Approved
   ```

2. **Notifikasi Kontak**
   ```
   Sistem → Tampilkan info kontak kedua belah pihak di dashboard
   ```

3. **Koordinasi Langsung**
   ```
   Pemilik/Penyewa → WhatsApp/Telepon → Tentukan waktu & lokasi
   ```

4. **Pickup Motor**
   ```
   Bertemu di lokasi yang disepakati → Serah terima motor
   ```

5. **Return Motor**
   ```
   Koordinasi lagi untuk pengembalian → Serah terima kembali
   ```

---

### **3. Dimana saya bisa melihat info kontak?**

**Jawaban:** Info kontak tersedia di dashboard masing-masing:

#### **Dashboard Pemilik Motor:**
- Section **"🚀 Booking Aktif"**
- Menampilkan info penyewa: nama, telepon, alamat
- Tombol quick WhatsApp & telepon

#### **Dashboard Penyewa:**
- Section **"🏍️ Rental Aktif"**  
- Menampilkan info pemilik: nama, telepon, alamat
- Tombol quick WhatsApp & telepon

---

### **4. Apakah alamat harus akurat sampai detail?**

**Jawaban:** **Tidak perlu detail rumah**, cukup area umum:

✅ **Yang Cukup:**
- "Cipayung, Jakarta Timur"
- "Dago, Bandung"
- "Malioboro, Yogyakarta"

❌ **Tidak Perlu:**
- Alamat lengkap dengan RT/RW
- Nomor rumah detail
- Kode pos

**Tujuan:** Hanya untuk gambaran area dan estimasi jarak koordinasi.

---

### **5. Bagaimana jika ada masalah saat koordinasi?**

**Jawaban:** Ada sistem backup:

🔧 **Mediasi Admin:**
- Hubungi admin melalui dashboard
- Admin bisa membantu komunikasi
- Akses data lengkap kedua belah pihak

📞 **Multiple Contact Methods:**
- WhatsApp (prioritas utama)
- Telepon langsung
- Through platform admin

🛡️ **Data Backup:**
- Semua info tersimpan di sistem
- History komunikasi
- Evidence untuk dispute resolution

---

### **6. Apakah bisa pickup di luar alamat yang terdaftar?**

**Jawaban:** **Ya, tentu saja!**

🎯 **Fleksibilitas Penuh:**
- Alamat di sistem hanya untuk **referensi area**
- Pickup/return bisa di **mana saja** sesuai kesepakatan
- Contoh: Alamat "Cipayung" tapi pickup di "Mall Cipinang"

💡 **Tips Koordinasi:**
- Tentukan landmark yang mudah
- Share live location saat bertemu
- Konfirmasi waktu kedatangan

---

### **7. Apa keuntungan sistem ini dibanding aplikasi lain?**

**Jawaban:** Beberapa keunggulan:

✅ **Fleksibilitas Maksimal**
- Pickup/return dimana saja
- Tidak terbatas lokasi tertentu
- Bisa nego waktu dan tempat

✅ **Personal Touch**
- Komunikasi langsung owner-renter
- Build relationship untuk repeat customer
- Trust building organik

✅ **Cost Effective**
- Tidak perlu infrastruktur pickup point
- Tidak ada biaya tambahan delivery
- Scale mudah ke mana saja

✅ **Keamanan Data**
- Info kontak hanya visible saat ada booking
- Data lengkap untuk keamanan
- Admin oversight untuk semua transaksi
